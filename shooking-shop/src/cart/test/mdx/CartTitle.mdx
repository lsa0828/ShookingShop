# CartTitle

장바구니 상단에 위치하여 **제목과 상품 개수**를 보여주는 컴포넌트입니다.  
현재 장바구니에 담긴 상품 수를 Recoil selector `cartNumSelector`를 통해 계산하며, 별도의 props 없이도 사용할 수 있습니다.


## 기능

- **제목**: `"장바구니"` 고정 텍스트
- **상세 설명**: `현재 N개의 상품이 담겨있습니다.` 형식으로 표시
- **장바구니에 담긴 상품 수**: `cartNumSelector`를 통해 Recoil 상태 기반으로 계산
